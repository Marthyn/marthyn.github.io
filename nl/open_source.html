Hours / Elmas / Hubot-praise

Hours

Hours is a project created by three of my former colleagues at Defacto. I did a some work on it. We all managed to get to 100% test coverage and a 4.0 on codeclimate. Mostly thanks to Jurre, Hugo and Matthijs. I did a few updates on it, to the graphs and facturation overviews. It was created because most time tracking apps are ugly and bloated. This is a simple no nonsense time tracking app that is easy to use and looks nice too.

Elmas

Elmas is a Rubygem I created by myself during my time at Hoppinger. It's an API wrapper for the Exact Online system. The challenge here was to set it up so resources would take minimal time to add, so the whole wrapper is very abstract and I used a lot of meta programming in it.

Hubot-praise

hubot-praise is a little script for the robot Hubot to praise a coworker or teammate. It was heavily inspired by a talk at Joy of coding. It allows you to send a praising message and an awesome GIF to someone else in your chat environment.

Hypatia

A gem you can use to determine the difficulty of a mathematical operation. It was used in a project I did where medical students had to learn how to calculate amounts of medicine. This app learned from the user's mistakes and made more difficult or easier questions based on their level.

Helpful

An awesome helpdesk system I did some minor work on. At Defacto we used it to track messages from people who had new ideas or wanted feedback or help. I am planning on doing more work for it and implementing a widget in this site.

New project(s)

I'm always up for a new open source project, I'm thinking of doing some work on Hawkthorne journey, an open source game based on a Community episode. But who knows, send me a message if you need help!
